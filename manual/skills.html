Skills
Each capability has many skills that can be learned in it.
Additional skills in a capability are unlocked by leveling up skills in that capability.
ÅüPhysical Skills
Skill Description
Speed Increases Celes's action speed in combat.
Recovery Increases the amount of HP Celes recovers after each round of combat.
Constitution Increase Celes's max HP.
Defense Increase Celes's ability to defend herself in combat.

ÅüWizardry Skills
Skill Description

Synergy Increase Celes's casting speed when casting spells from the same element in a row.
Magic Defense Increase Celes's defense against magical attacks.
