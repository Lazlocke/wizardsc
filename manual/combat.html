Celes will fight on her own in combat, but you can
assign her strategy and priorities for what skills
to use for the best chance of victory.
You can forbid her from using specific skills that
she's learned, or even assign skills as her special
attacks, only to be used when you give her the order
to go all out!
Change tactics on the fly when you come across strong
or dangerous enemies, but battle is ultimately up to her.
Be careful though. Running out of HP means that she'll
be forced to retreat and may spend multiple weeks
recovering from her wounds.

Physical Attack
Set Celes's priority to use physical attacks.
Attack Magic
Set Celes's priority to use attack spells.
Defense Magic
Set Celes's priorty to use defensive and recovery spells.
Targeting
Set which enemies Celes should focus on attacking.
Positioning
Set the distance Celes should try to keep from enemies.
Secret Skills
Set whether Celes is allowed to use skills specified as Secret Skills.
Retreat HP
Set how low Celes's HP can drop before trying to retreat.
Combat Speed
Set the game progression speed. It can be further increased by pressing the Ctrl key.
HP Display
Turn the HP display for monsters on/off.
